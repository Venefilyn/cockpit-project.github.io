Cockpit 173

by Martin Pitt – Jul 25, 2018
category: release

Cockpit is the modern Linux admin interface. We release regularly. Here are the release notes from version 173.

Storage: Offer installation of VDO

The Storage page now shows an “Install VDO support” link if the VDO package is not installed. This is similar to the on-demand installation of the NFS client packages in Cockpit 169.

Machines: Add disks to a virtual machine

The “Disks” tab of a virtual machine now has an “Add Disk” button. Using this new feature, it’s possible to create a virtual disk in a libvirt storage pool or attach an existing disk.

Thanks to Marek Libra!
