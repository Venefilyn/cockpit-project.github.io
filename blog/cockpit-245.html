Cockpit 245

by Allison Karlitskaya – May 26, 2021
category: release

Cockpit is the modern Linux admin interface. We release regularly.

Here are the release notes from Cockpit version 245 and cockpit-podman version 31.

Metrics: New PCP configuration dialog

The Metrics page header has a new “Metrics settings” button:

This allows you to enable or disable the Performance Co-Pilot (PCP) metrics collector:

Related options will be added in the near future.

Storage: Show both SHA256 and SHA1 Tang fingerprints

Newer versions of tang-show-keys will use SHA256 to compute the thumbprints of the keys, so Cockpit now shows both versions to allow key verification for both new and old versions.

Release: No more cockpit-cache tarball

The Cockpit project is no longer distributing the cockpit-cache tarball as part of its releases. This tarball previously contained the exact state of the node_modules/ directory at the time the release was made, and was provided for reproducibility and possible GPL compliance reasons (since the main release tarball contains a good deal of generated code).

This has been replaced with a separate git repository, cockpit-project/node-cache, which will now receive a tag for each release.

cockpit-podman: Added Korean translation

A new translation has been added for the Korean language.

Thanks to simmon for the efforts.
